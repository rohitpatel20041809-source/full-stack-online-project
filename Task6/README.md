# Task6 — Laundry Mart Login

Simple login page for a fictional "Laundry Mart" project. This folder contains a minimal HTML page and stylesheet for a two-column login layout with an image on the left and a login form on the right.

## How to open

- Open `index.html` in a browser (double-click or serve with a local static server).
- The page references `index.css` and expects an image file named `laundry.jpg` in the same folder.

## Files

- `index.html` — the HTML for the login page
- `index.css` — styles used by the page

## CSS Properties Used

Below are the selectors from `index.css` and the CSS properties used for each.

- `.container`: `height`, `display`
- `.left`: `width`, `background-color`
- `.left img`: `width`, `height`, `object-fit`
- `.form`: `width`, `background-color`
- `.form h1`: `text-align`, `margin-top`, `font-size`
- `.form p`: `text-align`, `font-size`, `padding-top`
- `.form label`: `margin-left`, `font-size`
- `.form input`: `padding`, `margin`, `width`, `height`, `border-radius`, `border`
- `.forgot`: `text-align`, `padding`, `margin-right`, `font-size`
- `button`: `margin-left`, `width`, `height`, `background-color`, `color`, `font-size`, `border-radius`, `margin-top`, `border`
- `.register`: `text-align`, `margin-top`

## Notes

- If `laundry.jpg` is missing, replace the image `src` or add an appropriate image in the folder.
- You can adjust widths, fonts, and spacing in `index.css` to fit different screen sizes.
