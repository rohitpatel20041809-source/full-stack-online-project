# Task 5 - Laundry Service Landing Page

## Overview

This project is a simple static landing page for a laundry service. It includes a navigation bar, hero section, service messaging, and a visual image to showcase the offering.

## Files

- `index.html` - Main HTML structure for the landing page.
- `index.css` - Styles for layout, navigation, hero text, and button.
- `laundry.jpg` - Hero image representing the laundry service.

## Features

- Responsive navigation menu with links for Home, Services, About Us, and Contact.
- Hero section with a headline, supporting text, and call-to-action button.
- Visual layout using inline-block columns for text and image.
- Clean styling with modern typography and accent colors.

## Usage

1. Open `index.html` in a web browser.
2. Review the landing page design and content.
3. Modify the text and styling as needed to fit your brand or service.

## Notes

- The page is built using plain HTML and CSS.
- There is no JavaScript or backend functionality included.
- The design is ideal for a simple service promotion or portfolio demo.
